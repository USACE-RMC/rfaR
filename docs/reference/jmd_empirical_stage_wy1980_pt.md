# JMD Empirical Stage-Frequency with Perception Threshold

Empirical stage-frequency results for JMD (WY 1980-2024) with perception
threshold applied.

## Usage

``` r
jmd_empirical_stage_wy1980_pt
```

## Format

A data frame with 54 rows and 3 columns:

- wy:

  Water year

- stage_ft:

  Stage (FT-NAVD88)

- plot_posit:

  Plotting position

## Examples

``` r
head(jmd_empirical_stage_wy1980_pt)
```
