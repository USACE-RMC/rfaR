# JMD RFA Results - Expected Only

JMD expected only stage-frequency results from RFA.

## Usage

``` r
jmd_rfa_expected
```

## Format

A data frame with 10000 row and 3 columns:

- AEP:

  Annual Exceedance Probabilities

- Expected:

  Expected stages corresponding to AEPs

## Examples

``` r
head(jmd_rfa_expected)
#>    AEP Expected
#> 1 0.98  3803.75
#> 2 0.97  3804.97
#> 3 0.96  3805.79
#> 4 0.95  3806.48
#> 5 0.94  3807.10
#> 6 0.93  3807.69
```
