# JMD Stage (WY 1980-2024)

Daily stage data from Water Year 1980-2024.

## Usage

``` r
jmd_wy1980_stage
```

## Format

A data frame with 29,858 rows and 4 columns:

- timestep:

  Timestep index

- date:

  Date

- time:

  Time

- stage_ft:

  Reservoir stage (FT-NAVD88)

head(jmd_wy1980_stage)
