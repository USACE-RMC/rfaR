# Cherry Cricket Dam Initial Elevation

Initial/starting elevation (FT-NAVD88) for Cherry Cricket Dam reservoir
routing. This elevation was used in HEC-HMS.

## Usage

``` r
cc_init_elev
```

## Format

A single numeric value representing the initial reservoir elevation
(FT-NAVD88).

## Examples

``` r
sapply(cc_init_elev,class)
#> [1] "numeric"
head(cc_init_elev)
#> [1] 5565
```
